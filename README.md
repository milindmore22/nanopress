# NanoPress
NanoPress is an experimental WordPress plugin that enhances content management by leveraging the browser's built-in AI APIs. It offers features such as content summarization and translation to help users manage their posts more efficiently.

## Features
- **Content Summarization**: Automatically generate summaries of posts using the browser's Summarizer API.
- **User-Friendly Interface**: Easy-to-use interface for managing posts and summaries.
- **Customizable Settings**: Adjust plugin settings to fit your content management needs.
- **Lightweight and Fast**: Optimized for performance to ensure quick loading times.
- **Compatibility**: Works with the latest versions of WordPress.
- **Open Source**: Available on GitHub for contributions and improvements.
- **Documentation**: Comprehensive documentation available for users and developers.

## Installation
1. Download the plugin from the WordPress plugin repository or GitHub.
2. Upload the plugin files to the `/wp-content/plugins/nano-press/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Configure the plugin settings as needed.

## Usage
### Summarization
- Navigate to the NanoPress settings page in the WordPress admin dashboard.
- Under Settings->NanoPress, you can enable or disable the summarization feature.
- Use the summarization feature on any post or page by clicking the "Summarize" button in the editor.
- The generated summary will be displayed in the designated output area.

## Translations
- The plugin supports translations on the fly using Translator API.
- To Enable translations, go to the NanoPress settings page and select your preferred languages.
- The plugin will automatically translate content based on the selected languages.
